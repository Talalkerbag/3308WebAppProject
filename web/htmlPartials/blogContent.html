Data Model

The Data Model has four tables. A User, Problem, Solution and User Role tables. Each user has a "Role"; either Admin, Normal User or Guest. Each User can create a problem and provide a solution for any other users to see or could wait for another user to solve the problem for them. The Solution Table connects both the User and Problem Table.
